# Nova E-Commerce Content Factory

AI-powered content generation demo built for the **Amazon Nova Hackathon**.

## Features

- Upload product image or video
- AI processing timeline visualization
- Marketing asset generation
- Caption and ad copy creation
- Premium product visual concepts

## Tech Stack

Frontend
- React
- JavaScript
- CSS

AI
- Amazon Nova
- Amazon Bedrock

## Demo Flow

1. Upload product asset
2. AI analyzes product
3. Background concept generated
4. Caption and ad copy produced
5. Marketing-ready assets displayed
